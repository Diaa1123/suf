# 🎯 كيف تضيف مكونات سُفرة للصفحة الرئيسية - دليل مفصل

## ⚠️ مهم جداً: المكونات الآن موجودة في الثيم!

المكونات التي أنشأناها (Hero و Products Grid) **موجودة في الكود** و **مسجلة في twilight.json**، لكن لكي تظهر في المتجر، يجب تفعيلها.

## 📋 الطريقة الصحيحة (حسب توثيق سلة الرسمي)

### الخطوة 1: التأكد من نشر الثيم ✅

```bash
# في Terminal
cd c:\Users\diaal\Downloads\suf

# بناء الثيم
npm run production

# الربط مع سلة (إذا لم يتم)
salla theme link

# النشر
salla theme publish
```

---

### الخطوة 2: الدخول للمتجر التجريبي 🏪

1. اذهب إلى [Salla Partners Portal](https://salla.partners/)
2. من القائمة اليسرى، اختر **"Stores"** أو **"المتاجر"**
3. اختر المتجر التجريبي (Demo Store)
4. اضغط **"Dashboard"** للدخول لوحة التحكم

---

### الخطوة 3: إدارة الصفحة الرئيسية 🏠

#### **الطريقة الأولى: من إعدادات المتجر**

1. في Dashboard المتجر، اذهب إلى:
   ```
   الإعدادات → التصميم → إدارة الصفحة الرئيسية
   Settings → Design → Homepage Management
   ```

2. ستجد قائمة بالمكونات المتاحة

3. ابحث عن مكونات سُفرة:
   - **"القسم الرئيسي (Hero) - سُفرة"** 🍽️
   - **"قسم المنتجات (Grid/List/Menu) - سُفرة"** 🍕

4. اضغط على أي منها، خصص الإعدادات، ثم احفظ

#### **الطريقة الثانية: من خلال تحرير الصفحة مباشرة**

1. اذهب إلى الصفحة الرئيسية للمتجر
2. إذا كان لديك صلاحية، سترى زر **"تحرير الصفحة"** أو **"Edit Page"**
3. اضغط عليه
4. اضغط **"+ إضافة قسم"**
5. اختر المكون المطلوب

## 🔍 إذا لم تظهر المكونات الجديدة

### السبب المحتمل 1: الثيم لم يُنشر بشكل صحيح

**الحل:**
```bash
# تأكد من التحديثات
git add .
git commit -m "Add Sufrah components"
git push origin main

# أعد نشر الثيم
salla theme publish
```

### السبب المحتمل 2: المكونات في twilight.json بشكل خاطئ

**التحقق:**
```bash
# تأكد من صحة الـ JSON
node -e "console.log(JSON.parse(require('fs').readFileSync('twilight.json', 'utf8')).components.filter(c => c.key.includes('sufrah')))"
```

يجب أن ترى:
```json
[
  { "key": "hero-sufrah-2026", "path": "home.hero", ... },
  { "key": "products-grid-sufrah-2026", "path": "home.products-section", ... }
]
```

### السبب المحتمل 3: الملفات غير موجودة

**التحقق:**
```bash
# تأكد من وجود ملفات المكونات
ls src/views/components/home/hero.twig
ls src/views/components/home/products-section.twig
ls src/views/components/product/card-restaurant.twig
```

يجب أن تظهر جميعها.

## 🔧 استكشاف الأخطاء

### خطأ: "Component not found"
**السبب:** المسار في twilight.json غير صحيح
**الحل:**
```json
// في twilight.json، تأكد من:
"path": "home.hero"  // ✅ صحيح
"path": "components.home.hero"  // ❌ خطأ
```

### خطأ: "Template not found"
**السبب:** ملف .twig غير موجود
**الحل:**
```bash
# تأكد من المسار الصحيح
src/views/components/home/hero.twig  # ✅
src/views/home/hero.twig  # ❌
```

### خطأ: المكون يظهر فارغ
**السبب:** لم يتم تخصيص الإعدادات
**الحل:**
1. اذهب لإعدادات المكون
2. أضف صورة، عنوان، زر
3. احفظ

## 🚀 الخطوات (5 دقائق فقط)

### الخطوة 1️⃣: تأكد من النشر (دقيقة واحدة)

```bash
# في Terminal
cd c:\Users\diaal\Downloads\suf

# بناء نهائي
npm run production

# نشر للمتجر
salla theme publish
```

**انتظر** حتى ترى:
```
✅ Theme published successfully!
```

---

### الخطوة 2️⃣: افتح المتجر (30 ثانية)

1. اذهب إلى: https://salla.partners/
2. اضغط **"Stores"** من القائمة اليسرى
3. اختر متجرك التجريبي
4. اضغط **"View Store"** أو **"Dashboard"**

---

### الخطوة 3️⃣: إدارة الصفحة الرئيسية (دقيقتين)

في Dashboard المتجر:

```
الإعدادات → التصميم → الصفحة الرئيسية
Settings → Design → Homepage
```

أو:

```
المظهر → إدارة المحتوى → الصفحة الرئيسية
Appearance → Content Management → Homepage
```

---

### الخطوة 4️⃣: إضافة Hero Section (دقيقة واحدة)

1. اضغط زر **"+ إضافة قسم"** أو **"Add Section"**

2. ستظهر قائمة بالمكونات المتاحة

3. **ابحث عن:**
   ```
   🍽️ القسم الرئيسي (Hero) - سُفرة
   Hero Section - Sufrah
   ```

4. اضغط عليه

5. **خصص الإعدادات:**
   - 📸 ارفع صورة خلفية (أي صورة طعام)
   - ✍️ اكتب عنوان (مثل: "أشهى الأطباق")
   - 📝 اكتب نص فرعي
   - 🔘 اكتب نص الزر (مثل: "اطلب الآن")

6. اضغط **"حفظ"**

---

### الخطوة 5️⃣: إضافة Products Grid (دقيقة واحدة)

1. اضغط **"+ إضافة قسم"** مرة أخرى

2. **ابحث عن:**
   ```
   🍕 قسم المنتجات (Grid/List/Menu) - سُفرة
   Products Section - Sufrah
   ```

3. اضغط عليه

4. **خصص الإعدادات:**
   - 📦 اختر منتجات (اختر 4-8 منتجات)
   - 📐 اختر نمط العرض:
     - **Grid** → للمخابز والحلويات
     - **List** → للتفاصيل الكاملة
     - **Menu** → للمطاعم الكلاسيكية

5. اضغط **"حفظ"**

---

### الخطوة 6️⃣: شاهد النتيجة! (30 ثانية)

1. ارجع للصفحة الرئيسية للمتجر
2. **اضغط F5** لتحديث الصفحة
3. **يجب أن ترى:**
   - Hero Section في الأعلى 🎉
   - Products Grid بالأسفل 🎉

---

## ❓ لم تظهر المكونات؟

### تحقق من هذه النقاط:

#### ✅ هل المكونات موجودة في twilight.json؟
```bash
# في Terminal
cd c:\Users\diaal\Downloads\suf
grep -A 3 "hero-sufrah" twilight.json
```

يجب أن ترى:
```json
"key": "hero-sufrah-2026",
"title": {
  "ar": "القسم الرئيسي (Hero) - سُفرة"
}
```

#### ✅ هل ملفات Twig موجودة؟
```bash
ls src/views/components/home/hero.twig
ls src/views/components/home/products-section.twig
```

يجب أن تظهر بدون خطأ.

#### ✅ هل تم البناء بنجاح؟
```bash
ls -lh public/app.css
```

يجب أن ترى حجم ~600KB

#### ✅ هل تم النشر بنجاح؟
```bash
salla theme publish
```

يجب أن ترى:
```
✅ Theme published successfully!
```
